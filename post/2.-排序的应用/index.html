问题一（荷兰国旗问题）

  • 给定一个数组arr，和一个数num，请把小于num的数放在数组的 左边，等于num的数放在数组的中间，大于num的数放在数组的右边。 要求额外空间复杂度O(1)，时间复杂度O(N)
/*定义两个变量L, R,L代表小于num的区域,R是大于num的区域,从头遍历数组,
看到比num小的与L区域的最后一个交换,看到比num大的,与R区域最小一个交换,遇到等于num的,跳过...	*/
public static int[] partition(int[] arr, int l, int r, int num) {
	int less = l - 1;
	int more = r + 1;
   	int cur = l;
	while (cur < more) {
		if (arr[cur] < num) {
			swap(arr, ++less, l++);
		} else if (arr[cur] > num) {
            swap(arr, --more, cur);
		} else {  //==num
			cur++;
		}
	}
	return new int[] { less + 1, more - 1 };//返回等于区域
}
public static void swap(int[] arr, int i, int j) {
	int tmp = arr[i];
	arr[i] = arr[j];
	arr[j] = tmp;
	}

问题二: 快速排序

  • 用荷兰国旗问题来改进快速排序, 把小于和大于num的区域拿去递归,中间部分不动 时间复杂度O(N*logN)，额外空间复杂度O(logN)
//经典快排
public static void quickSort(int[] arr, int l, int r) {
	if (l < r) {
		//swap(arr, l + (int) (Math.random() * (r - l + 1)), r);
		int[] p = partition(arr, l, r);	//利用下面荷兰国旗函数,把arr分为两块
		quickSort(arr, l, p[0] - 1);//边界
		quickSort(arr, p[1] + 1, r);//边界    
	}
}
//默认最后一个数作为划分
public static int[] partition(int[] arr, int l, int r) {
	int less = l - 1;
	int more = r;
	while (l < more) {
		if (arr[l] < arr[r]) {
			swap(arr, ++less, l++);
		} else if (arr[l] > arr[r]) {
			swap(arr, --more, l);
		} else {
			l++;
		}
	}
	swap(arr, more, r);
	return new int[] { less + 1, more };
}

public static void swap(int[] arr, int i, int j) {
	int tmp = arr[i];
	arr[i] = arr[j];
	arr[j] = tmp;
}
  • 快排的问题: 时间复杂度随着数据的状况不同而不同, 例如数组是一个有序数组,则它的时间复杂度是O(N^2), 别的状况下是O(Nlog(N));
  • 所以引入一个随机快排, 就是随机选一个数去划分数组的两遍
  • 就是在上面的代码上加上swap(arr, l + (int) (Math.random() * (r - l + 1)), r);产生一个随机数,然后用这个随机数去划分整个数组.

堆排序

  • 建立大根堆的时间复杂度为O(N)

  • 时间复杂度O(N*logN)，额外空间复杂度O(1)

  • 堆结构非常重要(一颗完全二叉树)

    • 1，堆结构的heapInsert与heapify
    • 2，堆结构的增大和减少
    • 3，如果只是建立堆的过程，时间复杂度为O(N)
    • 4，优先级队列结构，就是堆结构
public static void heapSort(int[] arr) {
	if (arr == null || arr.length < 2) {
		return;
	}
	for (int i = 0; i < arr.length; i++) {// 堆排序中,建立大根堆
		heapInsert(arr, i);
	}
	int heapSize = arr.length;
	swap(arr, 0, --heapSize);//最后一个数和堆顶交换,,然后大小减一
	while (heapSize > 0) {
		heapify(arr, 0, heapSize);
		swap(arr, 0, --heapSize);
	}
}
	//插入一个数,然后,调整为大根堆
public static void heapInsert(int[] arr, int index) {
	while (arr[index] > arr[(index - 1) / 2]) {
		swap(arr, index, (index - 1) / 2);
		index = (index - 1) / 2;
	}
}
//假设数组中有一个变化了,然后往下沉,再调整为大根堆
public static void heapify(int[] arr, int index, int heapSize) {
	int left = index * 2 + 1;//左孩子下标
	while (left < heapSize) {
		int largest = left + 1 < heapSize && arr[left + 1] > arr[left] //左孩子和右孩子谁大
            				? left + 1 
            				: left;
		largest = arr[largest] > arr[index] ? largest : index;
		if (largest == index) {
			break;
		}
        //还要向下沉
		swap(arr, largest, index);
		index = largest;
		left = index * 2 + 1;
	}
}

public static void swap(int[] arr, int i, int j) {
	int tmp = arr[i];
	arr[i] = arr[j];
	arr[j] = tmp;
}

在一个输出流中,求流出数据的中位数

  • 思路: 建立两个堆, 一个大根堆,一个小根堆, 然后每当插入数据时,

  • 堆排序可能很好的解决这个问题（这里假设大家知道如何在堆中进行插入和删除）。我们维护两个数组和一个变量mid_value。两个数组分别是小于mid_value的元素组成的的大顶堆（即堆顶是最大的元素）和大于mid_value的元素组成的小顶堆。如果新来的数字比mid_value大，则将其插入大于mid_value的最小堆。否则，插入最大堆。 case<1> 如果此时大于mid_value的元素比小于mid_value的多两个，那么将mid_value插入小于mid_value的最大堆，将大于mid_value的最小堆的最小值赋给mid_value，并且删除该最小堆的堆顶。 case<2> 如果此时小于mid_value的元素比大于mid_value的元素多。那么将mid_value插入大于mid_value的小顶堆中，将小于mid_value的大顶堆堆顶赋给mid_value，并且删除该堆顶。

    插入第k个元素，堆的插入和删除操作的复杂度（不管是比较次数还是移动次数）都是O(logk)<=O(logn)，这样总的时间复杂度就是O(n*logn)。
